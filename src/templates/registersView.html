Import Registers Table

To import a file, it must be .txt. The records should be in binary, meaning the file must contain 32 lines, where each line has 32 bits.

Import Data in Memory Table

To import a file, it must be .txt. You can place in it ONLY the memory addresses that you want to import (every 4 bytes and in hexadecimal format), in front of them after the ":" you will place the corresponding 32 bits

  • 4 :xxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxx
  • 8 :xxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxx
  • 1c:xxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxx
  • 20:xxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxx
  • 2c:xxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxx

Filter Data in Register Table

In the table you can search for record names, values ​​in decimal, binary, unsigned, and hexadecimal formats

  • There is only one rule, if you are looking for hexadecimal, the number must start with 0x

Filter Data in Memory Table

In the memory table you can search for any value in the table, such as an address or values.
